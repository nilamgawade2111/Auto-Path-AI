# Car Service

React application for the Car service.

## Development

### Using Tilt (Recommended)

```bash
tilt up
```

Access at: http://car.localhost

### Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- React 18.2
- Vite 5.0
- Tailwind CSS 3.4
- React Router DOM 7.10
- Styled Components 6.1
